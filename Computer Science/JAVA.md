## 데이터 타입
```
byte(1), short(2), int(4), long(8), float, double, string, void

아 존나많다. 왜 이렇게 많지

이게 예약어라고 쉽게 말해서 선언하는거임

나 short인데 니 컴퓨터 메모리공간 2byte만 쓸게~

그러면 컴퓨터가 할당해줌

근데 내가 실제로 쓰는 공간이 8byte인데 short선언하면 어떻게 됨?

오류남.

2byte만 쓰겠다는거는 숫자를 -32,768 ~ 32,767 이정도 범위에서 쓰겠다는 거랑 같음

실제로 전세계 인구가 70억명이면 뭘 써야 될까.

int는 숫자범위가 +-10억 정도임

그러면 long을 쓰면됨. 범위는 902경 3조 3370억 2036만 8547
```

## 배열(Array)
```
배열은 메모리로 구현한다.

실제 미리 할당받은 메모리가 배열이다.

직접 구현도 가능하다.

이 배열로 구현한게 리스트다

배열은 공간이 찰때까지 쓸 수 있다.

반면 리스트는 꽉차면 알아서 공간을 컴퓨터한테 할당받는다.

배열과 리스트 차이는 안정성 vs 유연함.

Array, List, Map, Hash 등 자료구조 라고 하는 애들은 별거 없다

데이터 저장을 어떻게 할 것인가에 대한 답이 될 유용한 도구다.
```
